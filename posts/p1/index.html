This blog is made thanks to amazing free software and services #

As a very first post for this blog, I need to thank the makers of the amazing software and services that make this blog possible.
Since they don't make money with those, the least I can do, is to mention and link them.

The aim of this blog is to be very fast and lean, leading to very high performances.
It's written in VSCode, rendered via a Static Site Generator (eleventy), built via npm on a Ubuntu server, pushed to a github repo, hosted via GitHub pages, and cached by Cloudflare CDN.

There will never be any advertisement or bloatware.

I hope I'll able to share with you information that is helping you somehow reach your goals!
